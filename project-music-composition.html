AI-Powered Music Composition

A deep dive into the world of generative AI to create original musical pieces from scratch.

View on GitHub →

Project Overview

This ongoing project explores the intersection of artificial intelligence and creativity. The primary goal is to develop a generative model capable of composing novel music. This involves extensive work in data preprocessing of MIDI and audio datasets, experimenting with various neural network architectures like LSTMs and Transformers, and fine-tuning models to understand musical theory, harmony, and rhythm. The ultimate aim is to produce music that is not only technically sound but also emotionally resonant.

Technologies Used

Python TensorFlow/Keras Librosa MIDI Generative Models

Challenges & Learning

One of the main challenges is curating a high-quality, diverse dataset that can teach the model a wide range of musical styles. Another significant hurdle is managing the long-term dependencies in music, which is crucial for creating coherent compositions. This project has been an incredible learning experience in advanced deep learning techniques and the nuances of audio data processing.

Current Status

The project is currently in the model development and fine-tuning phase. Early results are promising, with the model generating short, coherent melodies. The next steps involve scaling up the model and dataset to produce longer, more complex compositions.
